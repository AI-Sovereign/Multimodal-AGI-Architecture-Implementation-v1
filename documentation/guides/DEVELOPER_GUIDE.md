This guide details the steps required to install and set up the TCS-AGI Framework.

## 1. Prerequisites

* Python 3.10+ (Recommended)
* TensorFlow 2.15+
* A suitable NVIDIA GPU environment (recommended).

## 2. Installation

1.  Clone the repository or download the ZIP file.
2.  Navigate to the `/packages` directory.
3.  Install the package via pip:
    ```bash
    pip install .
    ```

## 3. Configuration

Review the core dependency and metadata settings in the **`resources/config/pyproject.toml`** file.
